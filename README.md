## Overview

This repository contains code, notebooks, and scripts from my undergraduate and graduate research — spanning Ghana, West Africa, Madagascar, Kentucky, and the U.S. Projects range from operational heatwave diagnostics and precipitation climatology to CMIP6 model validation and severe weather analysis.

---

## Projects

### Extreme Weather & Severe Storm Analysis
| Notebook | Description |
|----------|-------------|
| `1_EXTREME_PRECIP_PROBABILITY_KENTUCKY.ipynb` | Atlas-14-style extreme precipitation climatology using KY Mesonet data |
| `SPEI_1mon.ipynb` / `SPEI_U.S.ipynb` | Standardized Precipitation Evapotranspiration Index for drought monitoring |
| `SPI_1mon.ipynb` | Standardized Precipitation Index computation |
| `PREDICTION OF THE TROPICAL SYSTEMS.ipynb` | Tropical cyclone/system prediction diagnostics |

### Heatwave & Temperature Research
| Notebook | Description |
|----------|-------------|
| `Calculation of heatwaves over Ghana (Historical data).ipynb` | Historical heatwave detection and characterization using GMA station data |
| `TEMPERATURE VARIATIONS ACROSS GHANA.ipynb` | Spatiotemporal temperature trend analysis across Ghana |
| `anomaly.ipynb` | Temperature anomaly computation and visualization |
| `GDD_HISTORICAL.ipynb` | Growing Degree Days historical analysis |

### Precipitation & Monsoon Climatology
| Notebook | Description |
|----------|-------------|
| `MONSOON analysis.ipynb` | West African monsoon variability and dynamics |
| `CDS_API_PRECIPITATION.ipynb` | ERA5 precipitation retrieval via the Copernicus CDS API |
| `Performance_of_CMIP6_and_Gauge_data_in_selected_areas_in_GHana.ipynb` | CMIP6 vs. rain gauge validation over Ghana |
| `VALIDATION OF PRECIPITATION DATA FOR ACCRA/KUMASI.ipynb` | Station-based validation of gridded precipitation products |
| `PINKRAH (MADAGASCAR_ANALYSIS_OF_PRECIPITATION).ipynb` | Precipitation patterns across Madagascar |

### Air Quality & Atmospheric Composition
| Notebook | Description |
|----------|-------------|
| `NO2 Concentration in Kumasi and Accra.ipynb` | NO2 surface concentration analysis for Ghana's two major cities |
| `Box plot of air pollutants in Adventist SHS.ipynb` | Air pollutant distributions at a school monitoring site |
| `Oti_Landfill.ipynb` | Atmospheric impact analysis near the Oti landfill site |

### Climate Indices & Comfort
| Notebook | Description |
|----------|-------------|
| `Published_current_TCCI_colab-merged_AGCZ.ipynb` | Tourism Climate Comfort Index evaluation for Ghana (1961-2020) |
| `KNUST Boundary layer variations.ipynb` | Planetary boundary layer height and variability at KNUST |
